# MQST-Net

This repository is prepared for the implementation of **MQST-Net: A Multi-scale Four-Quadrant Spatio-Temporal Disentanglement Network for Multivariate Time Series Forecasting**.

The source code, dataset preprocessing scripts, experimental configurations, and running instructions will be released after the manuscript is accepted or upon publication.

## Paper

**Title:** Multi-scale Four-Quadrant Spatio-Temporal Disentanglement Network for Multivariate Time Series Forecasting

## Planned Release

The repository will include:

- Implementation of MQST-Net
- Local--Global Competitive Attention (LGCA)
- Branch-Relation Interaction (BRI)
- Dataset preprocessing scripts
- Dataset-specific configuration files
- Training and evaluation scripts
- Hyperparameter settings for reproducibility
